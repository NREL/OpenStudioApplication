OpenStudio Geometry Editor is an experimental feature, it is not yet reccomended for production use.

At this stage we are encouraging user to try out the tool and provide feedback to openstudio@nrel.gov.

Currently, the OpenStudio Geometry Editor cannot be used on OpenStudio Models with existing geometry developed outside of the Geometry Editor.

The suggested workflow is:
  1. Create a new OpenStudio Model.
  2. Import Space Types, Default Construction Sets, Thermal Zones, and other resources in the OpenStudio Application.
  3. Create a new floorplan in the Geometry Editor.
  4. Work through the model creating Spaces on each Story. Define the geometry for each Space by selecting the Space and using either the Rectangle or Polygon tool to draw. Note that each Space must have a single floor surface and Spaces on the same story cannot overlap.
  5. Assign Space Type, Default Construction Set, Thermal Zone and other properties to each Space and Story. You can create new Space Types, Default Construction Sets, Thermal Zones, and other resources using the library in the bottom pane.
  6. Use the preview buttom to preview your geometry in 3D.
  7. When you are satisfied with your geometry, use the merge button to apply your floorplan geometry to the current OpenStudio Model. Changes made to the floorplan in the Geometry Editor will not affect your current OpenStudio Model until you perform the merge step.
  8. Continue editing your OpenStudio Model in the OpenStudio Application; switching back to edit the floorplan using the Geometry Editor as needed. Each time you make a change to the floorplan you will need to re-merge with the current OpenStudio Model.
Current known issues are:
  • Placement of windows, daylighting controls, and other components is not yet available. These objects can be added via measures available on the BCL.
Thanks for giving the OpenStudio Geometry Editor a try, please let us know what you think!
